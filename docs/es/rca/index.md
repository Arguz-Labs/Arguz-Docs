# Analisis de causa raiz

Arguz RCA combina historial de deployments, contexto runtime y senales de politicas para acortar el camino desde el sintoma hasta la causa probable.

## Entradas usadas por RCA

- revisiones e imagenes
- alcance por cluster y namespace
- incidentes recientes
- contexto de nodos y workloads
- historial de ejecuciones de CronJobs cuando aplica
