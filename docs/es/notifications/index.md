# Notificaciones

Arguz separa la **creacion de canales** del **uso de canales en politicas**.

## Donde se crean los canales

Los canales de notificacion se crean y mantienen en la **Consola de administracion**, y luego se seleccionan desde:

- **Alert Policies**
- **Event Notification Policies**

Ruta de referencia:

`https://app-admin.arguz.io/admin/organizations/<organization-id>/notification-channels`

## Modelo practico

| Paso | Donde ocurre |
|---|---|
| Crear canal | Consola de administracion |
| Editar webhook o conector | Consola de administracion |
| Seleccionar canal para alertas | App principal |
| Seleccionar canal para eventos | App principal |
| Configurar horarios por canal | App principal |
