# Azure Active Directory (Azure AD)

Arguz currently uses **Google Identity Platform** as the primary authentication provider, supporting Google OAuth and email/password authentication.

## Azure AD Integration

Azure AD as an authentication provider is not currently available in the Arguz platform. The current authentication flow is described in the [Google Authentication](google-auth.md) documentation.

For organizations requiring Azure AD (Microsoft Entra ID) integration for single sign-on, please contact Arguz support to discuss your requirements. Enterprise SSO integrations are evaluated based on customer demand.

## Current Authentication Options

| Provider | Status |
|---|---|
| Google OAuth | Available |
| Email/Password (Google Identity Platform) | Available |
| Azure AD / Microsoft Entra ID | Contact support |
| Other SAML/OIDC providers | Contact support |

## Custom SSO

For enterprise customers needing custom Single Sign-On configurations, Arguz supports evaluating additional identity providers on a case-by-case basis. Contact your account representative to discuss your organization's requirements.
